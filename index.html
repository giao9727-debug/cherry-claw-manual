CherryClaw 智能体手册

运行在 CherryStudio 中的个人 AI 助手，集任务调度、多平台消息推送、技能扩展与持久记忆于一体。

📖 什么是 CherryClaw？

CherryClaw 是一个自主 AI 代理（Agent），运行在 CherryStudio 平台中。它能通过网络搜索、定时任务、技能扩展、多通道消息推送等能力，主动为你执行各种任务——不只是"你问我答"，而是能主动提醒你、定时检查、跨平台通知。

🤖

并非普通对话 AI

不仅能回复问题，还能主动执行任务、调度脚本、推送消息。

⏰

7×24 在线

定时任务（Cron）可在设定时间自动唤醒，无需手动触发。

🔌

可扩展技能

通过安装或编写技能（Skills）获得新能力，如小云雀 AI 创作。

⚡ 核心功能

⏰

定时任务 Cron

支持一次性或周期性任务。每天/每周/自定义 cron 表达式，自动执行你的指令。

🔔

主动通知 Notify

通过 Telegram、微信等渠道主动推送消息给你，即使你不在对话中也能收到。

🌐

网络搜索 Exa

集成搜索引擎，支持获取网页内容作为干净 Markdown，搜索实时信息。

🛠️

技能系统 Skills

从市场安装或自己编写技能，扩展 Agent 的能力边界。

🧠

持久记忆 Memory

FACT.md 存储长期知识，JOURNAL 记录事件。跨会话保留关键信息。

💬

多 IM 通道 Channels

支持微信、Telegram、飞书、QQ、Discord、Slack 等多种渠道接入。

技能是 CherryClaw 的能力扩展包。每个技能包含 SKILL.md 定义文件及脚本，让 Agent 获得特定领域的专业能力。

🧠 记忆系统

CherryClaw 拥有跨会话的持久记忆，不会每次对话都"重新认识你"。

📄

FACT.md

存储长期知识：活跃项目、技术决策、重要偏好。跨 6 个月以上仍有价值的信息放这里。

📋

JOURNAL.jsonl

记录一次性事件：已完成的任务、会话记录、重要笔记。每条带时间戳可搜索。

✅ 自动更新： 当 Agent 发现你的偏好或项目信息变化时，会自动更新 FACT.md。你也可以直接要求"记住这个配置"。

📡 多平台频道

CherryClaw 可以接入多种 IM 平台，通过你常用的聊天工具与你交互。

平台接入方式特点
💬 TelegramBot Token（从 @BotFather 获取）最稳定，支持频道/群组/私聊
💚 微信扫码连接本地微信客户端国内最常用，需微信桌面端运行
✈️ 飞书扫码自动创建应用或用 App ID/Secret企业用户友好
🐧 QQApp ID + Client Secret通过 QQ 开放平台接入
🎮 DiscordBot Token + OAuth2 邀请开发者社区常用
💼 SlackBot Token + App Token（Socket Mode）团队协作场景
📱 多通道协同： 可以同时连接多个平台，任务结果可以推送到指定频道或全部通知。

❓ 常见问题

CherryClaw 和普通 AI 聊天有什么区别？

普通 AI 聊天是"你问我答"；CherryClaw 是自主 Agent——可以定时执行任务、跨平台主动通知、安装技能获得新能力、持久记忆你的偏好。

怎么让 CherryClaw 在微信上跟我对话？

使用 add_channel 添加微信频道，扫码连接你的微信桌面客户端即可。之后微信消息会自动转发到这里。

技能安装后怎么用？

直接说需求就行。技能是按需自动触发的——比如装了 xyq-nest-skill 后，你说"做个视频"就会自动调用。

定时任务失败了怎么办？

Cron 任务有超时设置（默认2分钟），超时或失败会被自动中止。你可以增加 timeout_minutes 参数延长超时。

Agent 能记住我之前说过的话吗？

可以。跨会话的持久信息存储在 FACT.md 中，一次性事件记在 JOURNAL。但对话上下文在每次会话结束后会压缩，不会保留逐条聊天记录。

⚠️ 隐私提醒： CherryClaw 的技能和记忆存储在本地。如果需要使用联网功能（如搜索、小云雀 API），相关数据会经过对应服务商。
