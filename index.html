🧠 第一章：什么是 AI 智能体？

AI 智能体（AI Agent） 和普通聊天 AI 最大的区别在于——它能主动做事。

💬

普通 AI 聊天

你问一句，它答一句。对话结束就结束，不会主动帮你做任何事。就像一个知识渊博但从不主动说话的专家。

🤖

AI 智能体

能理解目标、拆解任务、调用工具、记住上下文。可以定时执行、主动通知、跨平台运行。

智能体的三大核心能力

能力说明举例
🛠️ 工具调用 能调用搜索引擎、执行代码、访问 API "查一下今天的天气" → 自动搜索天气网站
🧠 自主规划 能拆解复杂任务，分步执行 "帮我做一份市场调研" → 搜索→整理→出报告
💾 持久记忆 跨会话记住关键信息 你告诉过它的偏好、项目配置，下次记得
💡 直观理解： 普通 AI 像一本会说话的百科全书。AI 智能体像一个有手有脚的实习生——它能听懂你的目标，自己想办法完成，做完还会跟你汇报。

🎯 第二章：如何选择 AI 模型？

智能体的"大脑"就是大语言模型。选对模型，事半功倍。下面是 2026 年主流模型的选型指南。

模型能力金字塔

层级代表模型适合场景
🏆 旗舰Claude Opus 4.7, GPT-5.5, Gemini 3.1 Pro复杂编程、深度推理、专业级任务
⚡ 均衡Claude Sonnet 4.6, GPT-4o, Qwen3-VL日常开发、Agent 工作流、性价比之选
🚀 轻量Claude Haiku, Gemini Flash, DeepSeek Flash简单任务、高频调用、低成本
💰 极致性价比DeepSeek V4, GLM-4.5V, Qwen Flash中文场景、批量处理、预算敏感

需要看图/看视频怎么办？

不是所有模型都能处理图片。选择多模态模型时注意：

需求推荐模型说明
只看图 + 编码最强Claude Sonnet 4.6 / Opus 4.7看图精度最高，Agent 能力顶尖
图 + 音 + 视频全支持Gemini 3 Pro多模态天花板，2M 上下文
中文看图最佳GLM-4.6V / 豆包 Seed国产，中文识别优秀
开源 + 便宜Qwen3-VL-32B / MiMo V2.5Apache 2.0，可自部署
纯文本性价比DeepSeek V4 Flash最便宜的选项之一
⚠️ 选型原则： 不要用最强的模型做最简单的事。高频简单任务用轻量模型省钱，复杂任务上旗舰模型。混合使用最经济。

🏗️ 第三章：搭建你的智能体平台

目前主流的智能体平台是 CherryStudio（桌面端），它提供了一个完整的 Agent 运行环境。

快速上手指南

  1. 下载安装 CherryStudio —— 从官网下载并安装桌面客户端
  2. 配置模型 API —— 在设置中添加你的模型渠道（如 Claude、Gemini、DeepSeek 等）
  3. 创建智能体 —— 给智能体起名、选模型、写人设（SOUL.md）
  4. 配置环境变量 —— 如 XYQ_ACCESS_KEY、OPENAI_API_KEY 等
  5. 安装技能 —— 从技能市场安装或自己编写技能扩展能力
  6. 连接 IM 平台 —— 接入微信/Telegram 等，随时随地使用

智能体的三个配置文件

文件作用更新方式
SOUL.md定义智能体的性格、语气、原则对话中自然学习更新
USER.md记录你的偏好、时区、习惯对话中自然学习更新
FACT.md长期知识：项目信息、技术决策自动维护，也可手动要求记录
💡 最佳实践： 给智能体一个清晰的人设。比如"你是我的编程助手，喜欢用简洁的中文回答，优先考虑代码质量和安全性。"人设越清晰，智能体的表现越稳定。

📡 第四章：接入 IM 聊天平台

智能体的一大优势是能接入你日常使用的聊天工具，不需要每次都打开网页。

支持的平台与接入方式

平台接入难度最佳用途
💬 Telegram⭐⭐ 简单最稳定、功能最全，适合主力使用
💚 微信⭐⭐ 简单国内最普及，需桌面微信客户端配合
✈️ 飞书⭐⭐ 简单企业用户首选，可扫码自动创建机器人
🐧 QQ⭐⭐⭐ 中等需 QQ 开放平台注册应用
🎮 Discord⭐⭐⭐ 中等开发者社区常用，功能丰富
💼 Slack⭐⭐⭐ 中等团队协作场景，Socket Mode 接入

微信接入详解（最常用）

  1. 在智能体配置中选 添加频道 → 微信
  2. 系统会生成一个二维码图片
  3. 打开手机微信，扫一扫这个二维码
  4. 确认登录，微信就自动连接上了
  5. 以后在微信上给这个联系人发消息，智能体就能收到并回复
🔔 通知推送： 智能体可以主动给你发消息。比如定时任务完成后、监控到异常时，自动推送到你的微信/Telegram。

🔌 第五章：用技能扩展智能体

技能（Skills）是智能体的"插件"。每个技能赋予智能体一项专业能力。

技能能做什么？

🎬

AI 视频创作

说"做一段产品宣传视频"就能自动调用 AI 视频生成平台，上传参考图、编排工作流、下载成片。

📝

代码审查

自动审查代码变更，检查复用性、质量、效率问题，给出改进建议。

🔐

安全审查

对代码变更进行安全审计，发现潜在的安全漏洞和风险。

🔄

循环任务

定时执行特定任务，如每5分钟检查一次部署状态，自动汇报。

📅

定时调度

在指定时间自动触发远程任务，支持 cron 表达式。

➕

自定义技能

你自己也可以编写技能——写一份 SKILL.md 定义文件，配几个脚本，就能让智能体获得新能力。

技能的获取方式

  1. 从市场安装——搜索需要的技能，一键安装
  2. 从 Git 仓库添加——很多开源技能发布在 GitHub/Gitee 上
  3. 自己编写——用 init 创建目录 → 写 SKILL.md → 注册即可使用
📌 技能触发机制： 技能不需要手动开启。当你的请求匹配技能描述时，智能体会自动调用对应的技能。比如装了视频创作技能后，说"做个视频"就会自动触发。

⏰ 第六章：定时任务与自动化

让智能体在指定时间自动执行任务，是你最值得掌握的进阶功能。

支持的任务类型

类型说明示例
⏱️ 一次性任务在指定时间执行一次"30分钟后提醒我开会"
🔄 周期性任务按固定间隔重复"每2小时检查一次服务器状态"
📅 Cron 表达式灵活的时间规则"每天早上9点推送新闻" → 0 9 * * 1-5

实用场景举例

# 每日早报
"每天早上8点搜索今日科技新闻，推送到我微信"

# 股票监控
"每30分钟检查一下某只股票的价格，波动超过5%通知我"

# 周报提醒
"每周五下午5点提醒我写周报"

# 定时清理
"每天凌晨3点执行一次临时文件清理"
⏳ 超时说明： 定时任务默认超时时间为 2 分钟。如果是耗时较长的任务（如视频生成），可以指定更长的超时时间。

      ① 搜索多个信息源
      ② 整理成结构化摘要
      ③ 推送到你的聊天窗口

❓ 常见问题

🤔 智能体会泄露我的隐私吗？
所有记忆和配置文件存储在本地。联网功能（搜索、API调用）会经过对应服务商，请勿在对话中透露密码等敏感信息。
💰 用智能体是不是很贵？
看模型选择。如果全部用旗舰模型（Claude Opus、GPT-5.5）确实不便宜。但日常任务用性价比模型（DeepSeek Flash、Claude Haiku、Gemini Flash），一天几毛钱就够了。关键策略：简单任务用便宜模型，复杂任务上好模型。
🔌 技能从哪里来？
技能市场有现成的，GitHub/Gitee 上有开源技能，你也可以自己写。编写一个技能只需要创建一个 SKILL.md 描述文件和对应的脚本。
📱 手机能用吗？
通过微信/Telegram接入后，手机上直接发消息就能用，不需要开电脑。定时任务和通知推送也会直接到你手机。
🔄 智能体出错了怎么办？
检查模型渠道是否正常、环境变量是否配置正确、技能脚本是否有报错。大部分问题出在网络或配置上。
